평면 거울 가상 실험실

• 물체를 드래그하여 거울과의 거리를 조절해 보세요.
• 수평선의 px 수치를 통해 상의 위치를 확인하세요.
• 거울면을 기준으로 사과를 든 손의 위치 변화를 관찰하세요.
[실험 방법]
1. 빛의 경로 설정 버튼을 클릭하여 화면에 광선을 표시합니다.
2. 화면의 실험 물체를 잡고 좌우로 드래그하여 위치를 옮깁니다.
3. 수평선의 거리(px) 수치가 거울을 기준으로 항상 같은지 비교합니다.
4. 곰돌이의 사과를 든 손과 화살표의 방향이 반전되는지 관찰합니다.
