평면 거울 가상 실험실

• 물체를 드래그하여 거울과의 거리를 조절해 보세요.
• 배경의 방안지 눈금을 통해 상의 크기와 거리를 가늠해 보세요.
• 거울면을 기준으로 사과를 든 손의 위치가 어떻게 바뀌는지 확인하세요.
[실험 방법]
1. 빛의 경로 설정 버튼을 클릭하여 화면에 광선을 표시하고 물리적 원리를 확인합니다.
2. 화면의 실험 물체를 잡고 좌우로 드래그하여 거울에 가까이 가거나 멀어지게 합니다.
3. 배경의 눈금을 이용해 거울 속 상의 크기가 물체와 항상 같은지 직접 비교해 봅니다.
4. 곰돌이가 사과를 든 손의 위치와 수평 화살표가 가리키는 방향이 어떻게 반전되는지 관찰합니다.
